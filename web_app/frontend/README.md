# Frontend (React + LiveKit + Worker ONNX)

## Required env

```bash
VITE_API_BASE_URL=http://localhost:8000
VITE_API_WS_BASE=ws://localhost:8000
VITE_POSTURE_MODEL_URL=/models/best_posture_model.onnx
```

Copy `.env.example` to `.env` before running:

```bash
copy .env.example .env
```

## Scripts

- `npm run dev`
- `npm run build`
- `npm run lint`
